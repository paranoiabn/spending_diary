ДЛЯ КОГО? - современного человека, который хочет отслеживать свои траты, быть выделяющимся и эффективным в своей жизни. 
## Функциональность
- Загрузка данных (.csv) с банковских приложенией
- Ручное добавление/удаление трат
- Фильтрация / сортировка
- Валидация данных

## Стек
- Frontend: React, JS, Scss

## Установка и запуск
```bas
git clone https://github.com/paranoiabn/spending_diary.git
cd spending_diary
bun install
bun run dev
```
